# Kvrocks 🏃💨

[Kvrocks](https://github.com/KvrocksLabs/kvrocks) 是基于 RocksDB 之上兼容 Redis 协议的 NoSQL 键值存储服务。Zeno 将基于此提供高性能的键值存储服务，满足如加速缓存、计算中间层等场景。

Redis 命令兼容性详见 [Support Commands · Kvrocks](https://github.com/KvrocksLabs/kvrocks/wiki/Support-Commands)
