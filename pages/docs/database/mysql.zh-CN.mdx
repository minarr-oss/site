# MySQL

在日常的开发中，常常离不开数据库进行数据存储管理。因此 Zeno 为玩家提供了 MySQL 数据库，老样子：你只管使用就好（但简易管理方案目前在整理中，请耐心等待）。当然也可以选择使用外部数据库。

<Callout emoji="⚠️">
  当前功能基于 MySQL 8.0 的单机版实例提供。
</Callout>

为每位玩家免费提供三个存储量上限为 256MB 的 DB。请在**社群**中向 @Neo 申请实例。在拿到实例后，你可以先简单的通过代码实现 schema 和数据的管理，亦可委托 @Neo 进行手动 schema 增改。
