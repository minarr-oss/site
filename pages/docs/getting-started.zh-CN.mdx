# 入门

<Callout emoji="💡">
  Zeno 当前处于 Alpha 测试阶段，支持的框架和功能较为有限，各项功能正在超频开发中。
</Callout>

## 接入步骤

1. 安装 Zeno CLI
2. 注册账号或者登陆
3. 将 Zeno GitHub App 集成到你的 GitHub 账号
4. 在 Zeno 中创建一个新的项目
5. 使用分配的域名访问服务
6. （可选）绑定自定义域名并开始提供服务

## 安装 Zeno CLI

根据 [CLI - 安装与设置](/docs/cli/installation) 文档，安装 Zeno CLI。

## 注册与登录

初次使用的用户，可以使用 `zeno login` 开始帐号注册流程，可以使用邮箱注册或第三方平台（如 GitHub、Google 等）。

```
~# zeno login
Zeno CLI v1.00.86
Please visit the following URL in your web browser:
https://<i-am-the-login-url>

 ▄ Waiting login to be completed (4s)
```

根据提示在浏览其中打开注册/登录的页面即可进行流程。登录后可以使用 `zeno whoami` 查看当前登录的用户

```
~# zeno whoami
Zeno CLI v1.00.86
Current login as Nobody(nobody@zeno.so)
project limits: 1
```

> 注意：当前 Zeno 处于 Alpha 测试阶段，因此初始账号仅提供少量配额，如需更多配额请通过社群获取。

## 账号关联

**使用 GitHub 注册的用户可以忽略该步骤**

当前 Zeno 集成了 GitHub 作为用户代码仓库，如果没有选择 GitHub 作为注册渠道的话，将需要使用 `zeno login link` 选择 GitHub 进行账号关联。
关联后方可进行更多 GitHub 平台的操作。

```
~# zeno login link
Zeno CLI v1.00.86
✔ GitHub
Connecting account Nobody(nobody@zeno.so) with GitHub
```
