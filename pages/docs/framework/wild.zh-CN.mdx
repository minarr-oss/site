# 通用应用

## 构建设置

| 设置项                      | 默认值                | 说明                                       |
| :-------------------------- | :-------------------- | :----------------------------------------- |
| Root Directory              | .（以仓库为工作路径） | 定义构建的工作路径                         |
| Build Command（可选）       | 不执行                | 构建命令，留空代表不需要执行任何命令       |
| Artifacts Directory（可选） | 构建时的工作路径      | 该目录中的文件将作为最终运行时的所需上下文 |

## 运行时设置

| 设置项        | 默认值 | 说明                         |
| :------------ | :----- | :--------------------------- |
| Start Command | 空     | 服务启动命令                 |
| Service Port  | 空     | 最终提供服务的 HTTP 监听端口 |
