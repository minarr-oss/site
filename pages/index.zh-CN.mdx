Zeno 是为快速交付打造的应用分发平台。通过简单几个步骤初始化并关联代码仓库，便可在 Zeno 构建、部署并分发你的应用。

![](/workflow.jpeg)

**专注迭代，剩下的交给 Zeno 🙌**

> Zeno 当前 Alpha 测试中，SLO 也就不到 100 个 9，所以你懂（适当把玩）

推荐/常见部署应用：

- 博客、文档、大多数前端：常见博客程序、NodeJS（SSG、SSR）或者静态 HTML 页面
- API 服务：NodeJS、Golang

更多框架、场景将逐步支持
