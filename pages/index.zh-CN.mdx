Zeno 是为快速交付打造的应用分发平台，通过代码仓库联动快速的迭代，让开发者专注于应用迭代减少对分发的负担。

## 接入步骤

1. 安装 Zeno CLI
2. 注册账号或者登陆
3. 将 Zeno GitHub App 集成到你的 GitHub 账号
4. 在 Zeno 中创建一个新的项目
5. 使用分配的域名访问服务
6. （可选）绑定自定义域名并开始提供服务
